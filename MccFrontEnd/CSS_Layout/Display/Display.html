  • Pengertian Display

    CSS display adalah sebuah properti yang dimiliki oleh setiap elemen Html, dimana setiap tag/elemen html berada dalam sebuah kotak, properti display ini berfungsi untuk mengatur setiap perilaku kotak tersebut.

  • Value Display

    1. Inline : Elemen Html yang secara default tidak membuat baris baru, dan kotaknya akan mengikuti konten sehingga inline tidak dapat diatur ukurannya.

    2. Block : Elemen Html yang secara default akan membuat baris baru, dan kotaknya bisa diatur ukurannya.

    3. Inline-Block : Elemen html yang ukurannya bisa diatur tapi tidak akan membuat baris baru

    4. none : untuk menghilangkan sebuah elemen
